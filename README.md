# smart-flow-plugin介绍

基于IntelliJ IDEA平台开发的可视化流程编排插件，适用于Java SpringBoot项目开发。

## 功能特点

- 可视化流程设计
- 支持SpringBoot项目集成
- 图形化流程编排界面
- 支持自定义节点和连线

## 环境要求

- IntelliJ IDEA 2023.1.7+
- JDK 1.8+
- SpringBoot 2.x+

## 安装使用

1. 在IDEA插件市场搜索"smart-flow-plugin"并安装
2. 重启IDEA
3. 在工具栏找到插件图标即可使用
